# Privacy Policy — 운동/음악 블로그 에이전트

_최종 업데이트: 2026-07-06_

이 애플리케이션은 **개인이 본인의 기록을 정리하기 위해** 만든 개인용 도구입니다.
일반 대중을 대상으로 서비스를 제공하지 않습니다.

## 수집·이용하는 정보

- **Whoop 데이터**: 사용자가 직접 연결한 본인의 Whoop 계정에서
  운동(workout), 회복(recovery), 프로필 정보를 읽어옵니다.
  이 데이터는 블로그 글 초안을 생성하는 목적에만 사용됩니다.
- **사용자가 입력한 텍스트**: 운동 전후 기분·몸 상태 등 직접 작성한 내용.

## 저장·보관

- Whoop 인증 토큰은 앱이 실행되는 환경에만 저장되며, 제3자와 공유되지 않습니다.
- 생성된 글과 데이터는 사용자 본인만 접근하며, 외부에 자동 게시되지 않습니다.
  (네이버 블로그 게시는 사용자가 직접 복사·붙여넣기 하는 수동 방식입니다.)

## 제3자 제공

- 본 앱은 수집한 데이터를 판매하거나 광고 목적으로 제공하지 않습니다.
- 글 생성을 위해 입력 내용이 Anthropic(Claude API)로 전송되어 처리됩니다.
